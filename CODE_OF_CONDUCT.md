# Code of Conduct

This project welcomes contributors who want to improve the AI Powerlifting Form Analyzer.

## Expected Behavior

- Be respectful.
- Give constructive feedback.
- Keep discussions focused on the project.
- Help beginners when possible.
- Prefer clear technical explanations.

## Unacceptable Behavior

- Harassment
- Personal attacks
- Spam
- Abusive language
- Sharing private information without permission

## Reporting

If you notice unacceptable behavior, please report it through GitHub issues or by contacting the repository maintainer.
