# Security Policy

## Reporting a Vulnerability

If you find a security issue, please do not publicly exploit it.

Open a private report if GitHub security advisories are enabled, or contact the maintainer.

## Areas of Interest

Security-sensitive areas include:

- Authentication
- File uploads
- Video storage
- API authorization
- Environment variables
- MinIO credentials
- Database credentials

## Current Status

This project is in early development. Do not use it in production without a security review.
